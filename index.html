How moves work

Each move lets you pick one lit segment from any digit and place it into any empty segment slot of any digit. Intermediate shapes can be broken — only the final result must form valid digits.

The left-most digit is always pos0, then pos1, pos2… Up to 2 extra blank positions can be built into new digits within the move limit.

Segment labels (a – g)

aTop horizontal
bTop-right vertical
cBottom-right vertical
dBottom horizontal
eBottom-left vertical
fTop-left vertical
gMiddle horizontal

Example: 45 → 97 in 3 moves

Move 1: pos1(g) → pos0(a)

Move 2: pos1(d) → pos0(d)

Move 3: pos1(f) → pos1(b)

Starting number limited to 6 digits, move count to 6.
